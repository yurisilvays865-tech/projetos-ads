# Automação com Excel (Python + openpyxl)

Este projeto demonstra como criar e manipular planilhas Excel usando Python e a biblioteca openpyxl.  
Ele gera automaticamente uma planilha com dados organizados e formatados.

## Funcionalidades

- Criar uma planilha Excel (.xlsx)
- Criar cabeçalhos automaticamente
- Adicionar várias linhas de dados
- Salvar o arquivo no diretório local

## Tecnologias utilizadas

- Python 3
- openpyxl
