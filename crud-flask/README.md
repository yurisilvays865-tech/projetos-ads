# CRUD Flask

Este projeto é um exemplo simples de CRUD (Create, Read, Update, Delete) usando o framework Flask.  
O objetivo é demonstrar como criar rotas, manipular dados em memória e estruturar uma API básica.

## Funcionalidades

- Criar registros
- Listar registros
- Atualizar registros existentes
- Deletar registros
- Retornos em JSON para fácil integração

## Tecnologias utilizadas

- Python 3
- Flask
- Requests (para testes opcionais)
